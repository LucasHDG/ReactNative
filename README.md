## Getting Started

This is how you can launch the project from scratch.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/LucasHDG/ReactNative.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Launch the project
   ```sh
   npm start
   ```
